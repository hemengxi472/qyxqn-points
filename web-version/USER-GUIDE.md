## 系统简介

"青羊新青年"是中国移动青羊分公司青年员工能力积分管理系统。员工通过提交工作成果、参与公益活动等获取积分，管理员审核通过后发放积分。

**四大积分模块：**

| 模块 | 图标 | 说明 |
|------|------|------|
| 能力 | 📚 | 专业能力：获得公司/区级奖励、技术成果等 |
| 担当 | 🛡️ | 责任担当：承担课题研究、跨部门协作、问题上报等 |
| 道德 | ❤️ | 道德品质：文体活动、志愿服务、公益慈善、无偿献血、见义勇为等 |
| 纪律 | ⚖️ | 遵章守纪：仅通过团队任务获得，不可单独申请 |

## 员工端功能

### 注册与登录

**注册步骤：**

1. 打开系统地址，进入登录页
2. 点击"注册账号"切换到注册表单
3. 填写以下信息：
   - **用户名**：用于登录，建议使用工号或拼音
   - **密码**：至少 6 位
   - **姓名**：真实姓名
   - **工号**：员工工号
   - **部门**：所在部门名称
   - **头像**（可选）：上传个人头像照片
4. 点击"注册"，系统提示"注册成功，等待管理员审核"
5. 联系管理员审批通过后，即可使用注册的用户名和密码登录

> **注意：** 注册后默认为"待审核"状态，需等待管理员在后台审批通过后才能登录。

**登录操作：**

1. 在登录页输入用户名和密码
2. 点击"登录"
3. 员工登录后进入工作台首页（/dashboard）
4. 管理员登录后进入数据统计页（/admin/stats）

### 工作台（首页）

登录后首先看到的是工作台，包含以下信息：

**1. 总积分卡片**

页面顶部的大型粉色卡片展示当前总积分，数字会以动画效果滚动显示。

**2. 各模块积分**

以四张模块卡片展示每个模块的累计积分。点击卡片可进入积分申请页面。

| 模块 | 卡片颜色标识 |
|------|-------------|
| 能力 | 蓝色 |
| 担当 | 绿色 |
| 道德 | 金色 |
| 纪律 | 灰色 |

**3. 本月团队任务**

显示当前月份所属团队的信息：

- 团队名称
- 团队成员（以标签形式展示）
- 任务描述（由管理员统一设定）
- 任务状态（进行中 / 已提交 / 已通过 / 已驳回）

点击卡片可进入团队任务详情页。如果本月尚未分组，则显示"暂无团队任务"。

**4. 最近动态**

展示最近 5 条积分变动记录：

- `+` 绿色图标 = 积分增加
- `−` 红色图标 = 积分扣减

每条记录显示所属模块、子项、描述和分值。

### 积分申请

**操作流程：**

1. 在左侧菜单点击"积分申请"，进入模块选择页
2. 选择一个积分模块（能力 / 担当 / 道德），进入子项列表
3. 浏览该模块下的积分子项，每个子项显示：
   - 子项名称和描述
   - 每次可获得的分值
   - 每年可申请次数上限（如有限制）
4. 点击子项进入提交页面

**提交页面：**

1. 页面顶部摘要栏显示当前选择的模块、子项和默认分值
2. 在"描述说明"文本框中填写积分事项描述
   - 例如："2026年4月在成都市青羊区参加无偿献血"
3. 上传证明材料照片（必须上传，否则无法提交）
   - 点击上传区域选择照片，支持多张上传
4. 点击"提交申请"按钮

> **注意：**
> - 证明材料照片必须上传清晰可辨的图片
> - 部分子项有年度申请次数限制，超出后将无法提交
> - 提交后状态为"待审核"，积分不会立即到账

**提交后流程：**

```
员工提交 → 管理员审核 → 通过（积分到账）/ 驳回（不积分）
```

### 积分记录

在左侧菜单点击"积分记录"，查看所有积分变动明细。

- 每页显示 20 条记录
- 点击"加载更多"翻页
- 绿色 `+` 表示加分，红色 `−` 表示扣分
- 每条记录包含：模块名称、子项名称、描述说明、分值、时间

### 个人中心

在左侧菜单底部的用户区域点击进入个人中心。

显示信息：
- 头像、姓名、部门、工号
- 系统角色标签（员工 / 管理员 / 超级管理员）
- 登录账号

操作：
- 点击"退出登录"按钮，确认后退出系统

## 团队任务

团队任务是每月由管理员统一生成的分组任务。所有团队成员共同完成一项任务。

### 查看团队任务

1. 在工作台点击"本月团队任务"卡片，或在菜单点击"团队任务"
2. 查看团队信息：
   - 团队成员（列出所有成员姓名和部门）
   - 统一任务要求（管理员设置的本月任务描述）
   - 任务状态

### 提交团队任务

当团队状态为"进行中"时：

1. 点击"提交完成情况"按钮
2. 在弹出的对话框中填写完成描述
3. 上传相关照片（必须）
4. 点击"提交"

> **注意：**
> - 任意一位团队成员都可以提交，提交后全队状态变为"已提交"
> - 管理员审核通过后，**全队每位成员各获得 5 分纪律积分**
> - 如果被驳回，团队回归"进行中"状态，可重新提交
> - 在审核通过/驳回前，团队成员将看到"已提交，等待审核"的提示

### 历史团队任务

页面下方展示本月之前的历史团队任务记录，包含月份、团队名称和状态。

## 管理后台功能

管理员登录后自动进入后台，左侧菜单显示所有管理功能入口。

### 审核管理

**功能：** 审核员工提交的积分申请。

**操作步骤：**

1. 在菜单点击"审核管理"，进入审核队列
2. 页面顶部有三个标签页切换：
   - **待审核**：等待处理的申请
   - **已通过**：已批准的申请记录
   - **已拒绝**：已驳回的申请记录
3. 点击任意申请记录进入详情页

**审核详情页操作：**

1. 查看申请人的详细信息：
   - 姓名、部门、工号
   - 申请的模块和子项
   - 申请描述
   - 证明材料照片（点击可放大查看）
2. 在审核操作区：
   - 填写审核意见（可选）
   - 设置奖励分值（默认等于子项分值，可在 0-100 范围内调整）
   - 点击**"通过"**按钮：积分立即发放到申请人账户
   - 点击**"驳回"**按钮：申请被拒绝，不发放积分

> **审核要点：**
> - 仔细核对证明材料是否真实、完整
> - 描述内容是否与子项要求相符
> - 照片清晰可辨，与描述一致

### 注册审批

**功能：** 审批新员工的注册申请。

**操作步骤：**

1. 在菜单点击"注册审批"
2. 查看待审批列表，每张卡片显示：
   - 头像、姓名、部门、工号
   - 用户名、注册时间
3. 点击**"通过"**：员工状态变为"已激活"，可以正常登录使用
4. 点击**"驳回"**：员工状态变为"已禁用"，无法登录

### 数据统计

**功能：** 查看系统整体运营数据。

**统计卡片：**
- 在职员工数
- 总提交数
- 待审核数
- 作假记录数

**本月团队任务统计：**
- 已提交 / 已通过 / 已驳回 / 进行中 的团队数量
- 已提交待审核的团队列表，可直接点击"前往审核"

**模块积分分布：**
- 四个模块的积分累计柱状图，直观展示各模块积分占比

**积分排行榜 TOP 10：**
- 按总积分排名前 10 的员工
- 前三名以金/银/铜标识

### 员工管理

**功能：** 查看和管理所有员工账号。

**员工列表表格：**

| 列 | 说明 |
|----|------|
| 姓名 | 员工真实姓名 |
| 工号 | 员工工号 |
| 部门 | 所属部门 |
| 总积分 | 当前累计积分 |
| 角色 | 超级管理员 / 管理员 / 员工 |
| 状态 | 正常 / 已禁用（绿色/红色圆点） |
| 创建时间 | 注册时间 |

**操作（仅超级管理员可用）：**

- **设为管理员：** 将员工角色提升为管理员，获得后台管理权限
- **启用/禁用：** 切换员工账号的可用状态（禁用后无法登录）

### 团队管理

**功能：** 管理每月的团队分组和统一任务。

#### 设置统一任务

1. 在页面顶部的"本月统一任务"区域
2. 在文本框中输入本月所有团队需要完成的任务要求
   - 例如："4月联合社区开展反诈宣传活动，完成宣传海报制作和社区宣讲"
3. 点击"保存统一任务"

#### 生成本月团队

1. 点击"生成本月团队"按钮
2. 确认弹窗提示（系统将随机分配 3-5 人一组）
3. 系统自动生成团队，刷新列表

> **注意：**
> - 每月只能生成一次团队。如需重新分组，需先删除已有团队
> - 只有"在职"状态的员工会被分配进团队

#### 团队列表操作

- **查看团队详情：** 点击团队卡片，弹窗显示成员、任务、提交内容
- **审核团队提交：** 当团队状态为"已提交"时，在详情弹窗中点击"通过"或"驳回"
  - 通过：全队成员各获得 5 分纪律积分
  - 驳回：团队回归"进行中"状态
- **删除团队：** 点击垃圾桶图标，确认后删除团队及成员数据

### 模块管理

**功能：** 管理积分模块和积分子项（积分规则配置）。

#### 模块操作

- **新增模块：** 点击"新增模块"，填写模块名称、图标（emoji）、描述、排序
- **编辑模块：** 点击模块卡片右侧的"编辑"按钮
- **启用/禁用模块：** 切换开关控制（禁用后员工端不可见）
- **子项管理：** 点击"子项管理"展开该模块下的子项列表

#### 子项操作

在展开的子项管理区域：

- **新增子项：** 点击"新增子项"，填写以下信息：

| 字段 | 说明 |
|------|------|
| 子项名称 | 如"获得区级公司奖励" |
| 描述 | 子项的详细说明 |
| 积分值 | 每次申请可获得的积分（正整数） |
| 每年最多次数 | 0=不限，N=每年最多申请 N 次 |
| 需要上传照片 | 开启后提交时必须上传证明材料 |
| 排序 | 数字越小越靠前 |

- **编辑子项：** 点击子项右侧的"编辑"按钮
- **启用/禁用子项：** 切换开关控制

> **配置示例：**
>
> | 子项名称 | 积分值 | 年限次数 | 需要照片 |
> |----------|--------|----------|----------|
> | 无偿献血 | 5 | 2 | 是 |
> | 获区级表彰 | 10 | 3 | 是 |
> | 参加志愿服务 | 3 | 12 | 是 |
> | 获市级表彰 | 20 | 5 | 是 |
> | 见义勇为 | 30 | 5 | 是 |

### 作假管理

**功能：** 处理积分作假行为，执行积分清零。

#### 举报作假

1. 点击顶部"新建"区域（如已收起则展开）
2. 选择员工作假人员（下拉搜索）
3. 选择月份（格式：YYYY-MM，如 2026-04）
4. 填写作假原因
5. 点击"确认提交"

**系统自动处理：**
- 将该员工当月的积分全部清零
- 从总积分中扣减对应的各模块积分
- 生成作假记录

#### 作假记录表

| 列 | 说明 |
|----|------|
| 员工 | 作假的员工姓名 |
| 月份 | 被清零的月份 |
| 原因 | 作假原因 |
| 清零积分 | 被扣除的积分总数 |
| 操作人 | 执行操作的审核员 |
| 时间 | 操作时间 |

#### 撤销作假（超级管理员）

点击作假记录右侧的删除按钮，系统将：
- 删除该作假记录
- 重新计算该员工当月的积分
- 恢复被扣减的总积分

> **注意：** 撤销操作仅超级管理员可执行。

## 常见问题

### Q: 注册后无法登录？

A: 新注册账号默认为"待审核"状态，需等待管理员在"注册审批"中通过后才能登录。请联系管理员处理。

### Q: 提交申请后积分多久到账？

A: 积分在管理员审核通过后立即到账，无延迟。可在工作台和积分记录中查看。

### Q: 为什么某些子项无法申请？

A: 可能的原因：
1. 该子项已被管理员禁用
2. 已达到年度申请次数上限
3. 所属模块已被禁用

### Q: 证明材料照片有什么要求？

A: 请上传清晰、完整的证明材料照片，如献血证、获奖证书、活动照片等。模糊不清或无法辨认的照片可能导致审核不通过。

### Q: 纪律积分怎么获取？

A: 纪律积分无法单独申请，只能通过完成每月团队任务获得。管理员审核通过团队提交后，每位团队成员获得 5 分。

### Q: 团队任务由谁提交？

A: 团队中任意一位成员都可以提交。提交后全队状态变为"已提交"，其他成员无需重复提交。

### Q: 每个月可以生成几次团队？

A: 每月只能生成一次团队。如需重新分组，管理员需先删除现有团队再重新生成。

### Q: 密码忘了怎么办？

A: 请联系系统管理员重置密码。

### Q: 手机端怎么打开菜单？

A: 在手机浏览器中打开系统后，点击页面左上角的汉堡菜单按钮 (☰)，左侧菜单会以抽屉形式滑出。点击遮罩层或关闭按钮收起菜单。
